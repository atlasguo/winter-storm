# U.S. Winter Storm Visualization

An interactive map visualizing the NOAA Winter Storm Severity Index (WSSI) across the contiguous United States.

Optimized for desktop viewing.

**Data source:**  
NOAA Winter Storm Severity Index (WSSI)  
https://www.wpc.ncep.noaa.gov/wwd/wssi/wssi.php
